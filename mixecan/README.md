# Mixecan Investor Microsite

Single-page investor deck implemented with semantic HTML5, CSS3, SVG and Vanilla JavaScript. It is ready for static hosting on GitHub Pages.

## Included deck sections

- 01 — Cover
- 02 — Investment Opportunity
- 03 — Company Overview
- 04 — Product Portfolio
- 05 — Who We Are & What We Do
- 06 — Market Opportunity
- 08 — Financial Performance
- 09 — Why Invest in Mixecan

The official numbering intentionally preserves the gap for slide 07. The folder structure is ready to receive the remaining approved slides later without renumbering the existing sections.

## Navigation

Use the mouse, trackpad, touch gestures, or the arrow/Page Up/Page Down/Home/End keys.

## Publishing

Upload the contents of this folder to the root of a GitHub repository and enable GitHub Pages from the repository's default branch. No build step or package installation is required.
