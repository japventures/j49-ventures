# Mixecan Private Investment Microsite

This folder is designed to live at `/mixecan/` inside the repository used by `j49.mx`.

## Publish
1. Copy the entire `mixecan` folder into the root of the J49 GitHub Pages repository.
2. Commit and push.
3. Open `https://j49.mx/mixecan/`.

The page includes `noindex, nofollow` metadata, but anyone with the direct URL can access it. GitHub Pages does not provide true password protection.
