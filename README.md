# J49 Ventures Website

Static GitHub Pages website for `j49.mx`.

## Included routes

- `/` — J49 Ventures institutional landing page
- `/mixecanopportunity/` — direct, unlisted Mixecan opportunity page
- `/opportunity-template/` — reusable starting point for future direct landing pages

## Upload to GitHub

Upload the contents of this folder to the root of the `j49-ventures` repository. Keep the `CNAME` file in the root.

## Create a new direct landing page

1. Duplicate the `opportunity-template` folder.
2. Rename the folder using a clean URL slug, for example `project-atlas`.
3. Edit that folder's `index.html`.
4. Commit the changes.
5. The page will be available at `https://j49.mx/project-atlas/`.

Pages are not linked from the main website. However, an unlisted URL is not true access control. Use a secure data room or authentication for sensitive documents.
