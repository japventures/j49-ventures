# J49 Ventures — Phase 1 clean repository

This package is designed to replace the current contents of the GitHub Pages repository.

## Included routes

- `/` — J49 Ventures institutional homepage
- `/mixecan/` — temporary Mixecan holding page

## Upload procedure

1. Delete all files and folders currently in the repository, except the `.git` folder when working locally.
2. Copy every file and folder from this package into the repository root.
3. Commit and push to the branch used by GitHub Pages.
4. Confirm that GitHub Pages is publishing from the repository root.

The `CNAME` file must remain in the repository root so the custom domain continues to use `j49.mx`.
