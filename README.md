# J49 Ventures Website

Minimal institutional landing page for J49 Ventures, prepared for GitHub Pages and the custom domain `j49.mx`.

## Structure

- `/index.html` — main J49 Ventures landing page
- `/mixecanopportunity/` — direct, unlisted Mixecan opportunity page
- `/opportunity-template/` — reusable structure for future direct landing pages
- `/assets/` — shared images, CSS and JavaScript
- `/CNAME` — GitHub Pages custom-domain configuration

Direct pages are not linked from the homepage. They remain accessible to anyone who knows the exact URL and should not be treated as secure data rooms.
